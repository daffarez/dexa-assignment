# Frontend - Employee Monitoring System

Aplikasi monitoring karyawan, mencakup autentikasi, manajemen profil, absensi, notifikasi real-time.

## Setup Environment

```
VITE_SUPABASE_URL=
VITE_SUPABASE_KEY=
```

## Install Dependencies

```
npm install
```

## Run the App

```
npm run dev
```

App will run at `http://localhost:5173/`
